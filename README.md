# API Test Automation Framework

## Description
A RestAssured-based API test automation framework for testing JSONPlaceholder API endpoints.

## Tech Stack
- Java 17
- Maven
- RestAssured 5.3.0
- TestNG 7.7.1

## Test Coverage
- User CRUD operations (GET, POST, PUT, DELETE)
- Post management and validation
- Comment verification
- Multi-step workflows
- Positive and negative test scenarios

## How to Run
```bash
mvn clean test
```
